 # BinLockPro - 软件特色

 BinLockPro 是一款面向个人与小团队的桌面端效率工具，围绕「**文件加密/解密**、**隐私风险扫描**、**安全删除（粉碎）**」等高频需求，提供简单可靠的一站式解决方案。

 ## 核心亮点

 - **离线优先，隐私优先**

   - 加密/解密、扫描、粉碎等核心能力均在本地完成。

   - 默认不依赖云端，不上传你的文件内容。

 - **简单易用的文件加密 / 解密**

   - 面向日常场景：给文件上锁、分享前脱敏、对重要资料做备份保护。

   - 流程清晰：选择文件 -> 输入密钥 -> 开始加密/解密。

 - **隐私体检（敏感信息扫描）**

   - 支持对指定目录进行敏感信息规则扫描，帮助你发现“可能泄露隐私”的文件与内容线索。

   - 提供内容浅扫描模式：在本地读取少量文件头部进行匹配，降低扫描成本。

 - **安全文件粉碎（不可恢复删除）**

   - 支持粉碎文件/文件夹并递归处理。

   - 支持多种覆盖策略（不同强度的安全删除）。

   - 可选“敏感操作二次确认”，减少误删风险。

 - **更可靠的自动更新机制**

   - 内置自动检查更新；也支持手动检查、下载并重启安装。

   - 更新过程可视化（检查状态、下载进度、安装提示）。

 ## Pro 功能

 - **隐藏空间**

   - 将文件放入加密保护的“隐藏空间”，在同一软件内完成管理与导入导出。

 - **高级安全操作**

   - 支持更严格的二次确认策略与更安全的删除/处理方式。

 ## 适用人群 / 场景

 - **个人用户**：保护证件照、合同、账单、项目资料、私人照片等。

 - **开发者**：扫描并定位可能误提交的私钥、Token、配置文件等敏感内容。

 - **小团队**：对共享资料进行加密传输与本地安全删除，降低泄露风险。

 ## 关键词

 文件加密 / 文件解密 / 隐私体检 / 敏感信息扫描 / 安全删除 / 文件粉碎 / 离线工具 / 桌面端
